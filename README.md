# L2-projet-web

Ce dépôt containt le code source du projet de developpement web de L2. L'objectif est de créer un site en lien avec Pokemon en utilisant [PokeAPI](https://pokeapi.co/). Dans le cadre de ce projet, devait être implémentés les fonctionnalitiés suivantes :

- Un affichage des pokémons avec leurs caractéristiques.
- Un système de combat Pokémon.
- Un système de progression des pokémons (XP, niveaux, évolutions).
- Une gestion d'inventaire pour les objets.

## Lancement

Pour lancer le projet en local, suivez les étapes suivantes :

1. Vérifier que Node.js et npm sont installés sur votre machine.
2. Dans le terminal, utilser la commande `npm install` pour installer les dépendances.
3. Pour lancer le serveur de développement, utiliser la commande `npm run dev`.
4. Ouvrir votre navigateur et aller à l'adresse `https://localhost:5173` (HTTPS est requis) pour voir le site.

## Technologies utilisées

- **API**: [PokeAPI](https://pokeapi.co/)
- **Frontend**: HTML, CSS, JavaScript
- **Outils externes**: [Vite](https://vitejs.dev/) pour le bundling et le développement local.
- **Dépendances externes** : [qrcode](https://www.npmjs.com/package/qrcode) (génération de QR codes), [Tailwind CSS](https://tailwindcss.com/) (framework CSS) et [Lucide Icons](https://lucide.dev/) (icônes).
